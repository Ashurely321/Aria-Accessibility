Web accessibility means designing websites that everyone can use, extremely helpful for people with disabilities. It’s about making sure content is readable, navigable, and usable no matter how they're browsing; whether someone is using a mouse, keyboard, screen reader, or other assistive technology they will be able to experience the content.

ARIA (Accessible Rich Internet Applications) helps improve accessibility by adding extra information to HTML elements. It tells screen readers how parts of a page behave, especially when using custom components or interactive features that aren’t natively accessible. Everyone can experience the page how it was meant to be.
